## 一、节奏判断类
*用途：判断当下处于行情的哪个阶段，今天该不该动*

**规律16：量能台阶是行情性质的分水岭**（6.4早盘）
- 3万亿以下：存量博弈，轮动为主，不能追高
- 突破3万亿+并稳住：增量入场，行情升级，可适度加仓
- 量能不能上台阶而指数涨：大资金获利了结，行情透支

**规律D：量能台阶的两种走法**（5.11-13）
- 缩量但指数不破位 = 筹码稳定
- 放量下跌 = 分歧扩大
- 放量修复 = 资金继续进攻

**规律9：量能信号的两段式读法**（6.2复盘）
- 第一段：AI硬件放量 → 反弹未结束，继续持有
- 第二段：AI硬件也缩量 → 反弹接近结束，资金转向业绩/价值方向

**规律2：弱势量能逻辑**（6.2早盘）
- 放量 = 砸盘资金增加，头顶悬剑
- 缩量 = 企稳信号，等缩量+情绪回暖+利好落地再入场

**规律19：缩量调整=开盘安全牌**（6.5早盘）
- 追高资金锁仓/离场 → 成交腰斩 → 涨不上去但跌不下来
- 健康调整形态，不是趋势反转；判断标准：成交量 vs 前一日比较，腰斩+涨跌家数改善=良性

**规律20：第N交易日的历史节奏**（6.5早盘）
- 日线顶部结构形成后第17个交易日，按历史节奏接近尾声
- 不能作为加仓理由，仍以纪律线为唯一操作标准

## 二、行情性质判断类
*用途：识别当日下跌/上涨是哪种性质，决定防守还是进攻*

**规律14：外力扰动 vs 内生调整的辨别**（6.3复盘）
- 外力扰动：消息传闻+量化踩踏打破结构，主线未瓦解，后市延续内生逻辑
- 内生调整：主线情绪退潮、资金主动撤退，需等新逻辑
- 判断方法：看下午反弹质量 + 尾盘是否回收跌幅 + 第二天主线是否反包

**规律17："一九行情"的底层健康度检验**（6.4复盘）
- 表面看：极少数科技上涨，多数个股下跌
- 底层看：微盘股/全A若同步破位 → 底部支撑不牢，主线随时可能被拖累
- 正常上升期：大小指数可快慢不一，但不能一涨一跌；背离持续则风险上升
- 应用：盘后务必检查微盘股、全A，不能只看科技旗手

**规律18：补跌的两种结局**（6.4复盘）
- 好的补跌：隔夜美股已跌 → A股科技大幅低开 → 一次性消化，随后快速回流 → 低吸窗口
- 坏的补跌：美股未跌但A股科技主动下行 → 内部筹码松动，需减仓

**规律8：普涨共振 vs 强共振的辨别**（6.2复盘）
- 普涨共振：全线涨但无聚焦，宽度有余高度不足，反弹预期降低
- 强共振：有主线分支+可预期情绪载体，能打出高度，行情持续性强
- 判断标准：情绪龙头能否连板、能否稳定封板

**规律4：指数涨停家数背离信号**（6.2早盘）
- 涨停家数创年内新高但指数仍跌 → 机构股以外的多数个股已不愿再跌
- 含义：止跌企稳信号已现，底部可能临近

**规律3：机构股情绪脆弱期识别**（6.2早盘）
- 怀疑种子一旦种下，传导速度快于基本面
- 东山精密跌停 → 机构股跳水，是典型案例
- 风格切换期：非机构股/小市值/题材股为最大受益方向

**规律5：风格切换的节点识别**（6.1）
- 切换信号：曾经强势的板块开始分化回撤，曾经被压制的低位板块反而翻红
- 当前节点参考：AI科技调整 + 微盘/ST翻红 = 风格已从主线科技向低位题材漂移

## 三、选股与接力类
*用途：在确定方向后，具体怎么选标的、怎么判断能不能打*

**规律15：接力选股的"中军锚"原则**（6.3复盘）
- 选板块内接力龙头时，先看中军（板块第二梯队代表股）是否稳定向上
- 中军稳 → 龙头抛压快速消化 → 封板，接力胜率大增
- 案例：亨通光电接力前，长飞光纤持续向上是关键正反馈

**规律1：炒新不炒旧**（6.2早盘）
- 同一板块内，新热点永远 > 旧热点
- 验证：MLCC/超级电容 > PCB（科技内部）；煤炭 > 电力（防御内部）

**规律10：海外确定性 vs 国内预期的分层操作**（6.3日盘）
- 有海外大单+业绩落地：直接配置，确定性高
- 国内材料/零部件卡位：次优，等订单导入
- 国内纯替代/在研：博弈仓位，等流片/量产硬信号，不能追高

**规律6：AIPC的"低位+科技"双属性逻辑**（6.1）
- 既有产业催化（避免纯炒概念质疑），又无高位筹码压力（避免机构抛压）
- 这类交叉地带往往是风格切换期最佳藏身处

**规律C：上游材料的逆势规律**（5.11-13）
- 主线题材回调日，上游硬材料（陶瓷基板、电子材料、钻针）往往逆势走强
- 原因：供给刚性+景气度确定性比纯情绪题材强，调整日资金反而流入
- 应用：AI科技调整日优先盯铜缆、光纤、MLCC、超级电容、电子化学品

**规律21：风格切换的质量判断——超跌反弹 vs 产业趋势接棒**（6.7复盘）
- **低质量切换**：领涨方向是消费等超跌板块，无产业趋势承接，持续性差
- **高质量切换**：领涨方向是科技内部分流至有产业趋势的方向（机器人/智能体等），持续性明显更高
- 判断方法：看领涨方向是否有近期产业催化支撑（新品发布/政策/业绩）

**规律22：成交拥挤度警示**（6.7复盘）
- 主线成交占比进入40%-51%区间 → 历史上主线均进入**缩圈阶段**
- 表现：主线旗手从"超额收益"退化为"市场β"（如光模块类比2021年茅台）
- 应对：主线旗手减仓，寻找还没被充分交易的产业趋势分支

**规律23：破位后的纠错信号双条件**（6.8早盘）
- 破位后不靠感觉回补，需等待两个信号之一：
  ①**技术信号**：60分钟及以上周期形成明确底部结构
  ②**价格信号**：收盘突破并站稳关键阻力位
- 适用于任何"有纪律线的趋势交易"体系

**规律24：午后率先走强题材的惯性效应**（6.8早盘）
- 指数承压期，**午后率先强势的题材板块**往往形成短期惯性（液冷→机器人→?）
- 资金会反复尝试同一模式，直至亏钱效应出现
- 操作：盘后记录午后最先走强方向，次日早盘优先关注该方向延续性

## 四、板块与避险类
*用途：判断防御板块的性质和退出时机*

**规律7：避险板块内部的过渡性判断**（6.1）
- 电力 → 煤炭的切换，本质是"炒新不炒旧"在防御内部的再次验证
- 过渡性避险方向：只适合短线，不适合持有
- 见顶信号：后排无法晋级，前排拉出长上影线，加速一字板

## 五、产业趋势类
*用途：理解大方向的结构性逻辑，不被短期波动带偏*

**规律11：AI瓶颈传导链**（6.3日盘）
```
算力（GPU）→ 互联（DSP/CPO/光模块）→ 供电（燃气轮机/SOFC/电力基建）
```
每个瓶颈环节突破一次，催生一轮相关板块行情。当前：互联+供电两端同时发力。

**规律A：Token消耗的飞轮效应**（5.11-13）
- Token消耗增速远快于硬件扩产速度 → 算力持续供不应求
- "Token工厂"模式（GPU利用率最大化）是最受益的商业模式演进方向

**规律B：CPU复权逻辑**（5.11-13）
- AI从训练→推理转型，GPU:CPU配比从8:1压缩至1:1
- CPU从配角变主角，带动封测（通富微电）、内存接口（澜起科技）、CPU本体（海光信息）系统性受益
