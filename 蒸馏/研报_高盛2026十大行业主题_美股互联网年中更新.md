# 研报蒸馏｜高盛《10 Industry Themes for 2026 & Focus Stocks（年中更新）》

> 来源：Goldman Sachs Americas Technology: Internet & Interactive Entertainment，2026-06-22（Eric Sheridan 等）
> 主题：**美股互联网/科技**十大主题年中更新 + 焦点股
> 性质：参考研报。**注意：这是美股互联网，不是A股**。对A股的价值在于——它解释了**美股AI资本开支周期**，正是博主框架里"AI硬件受美股牵制、跟随纳指起落"的源头。

## 一、三大投资者核心辩论

1. **AI投资周期**：scale模型/算力的capex/opex、融资来源（近期大额融资）、回报可见度。
2. **全球消费者健康度**（除中国外）：数据走软 + 通胀压力 + 广告/线上线下竞争加剧。
3. **AI颠覆主题**：AI去中介化/颠覆风险，对消费平台、导流、内容创作分发的终值冲击（agentic commerce崛起）。

## 三、焦点股（正向风险收益，按6个月股价错杀程度）

> 逻辑：AI可见度/融资需求/周期性担忧压制了部分大市值龙头；高盛认为未来3-5年可复制**双位数营收增长 + 营业利润率上行 + 相对历史有吸引力的估值**。

| 标的 | 一句话逻辑 |
|---|---|
| **GOOGL**（Alphabet） | 跨AI主题定位最佳（消费/企业用例、算力基建、自研芯片、Search演化） |
| **AMZN**（亚马逊） | 几乎正向暴露所有长期主题（云/AI/广告/机器人/AV/全球通信） |
| **META** | 核心业务（Family of Apps）增长强、被市场低估；6-12月产品发布是催化 |
| **NFLX**（Netflix） | 复合营收增长+利润率上行+大额资本返还 |
| **UBER** | 全球出行+配送领导地位，多产品（Uber One） |
| **DASH**（DoorDash） | 本地商业战略执行强、平台经济改善 |
| **SPOT**（Spotify） | 平台创新+AI内容净受益者+利润率上行 |

## 四、两个宏观观察（影响美股科技情绪）

1. **渠道调研**：本季度只有少数收入敞口跑向上行——中东冲突（3月起）后广告品牌预算和可选消费更谨慎；**唯一持续超预期的是 Google Cloud / AWS 的超大规模收入 + AI对直效广告的提振**。
2. **股权融资潮**：Alphabet 近期约 **$850亿融资**——投资者现在认为大型高capex玩家在未来6-12月进一步融资是"何时"而非"是否"的问题，**潜在股权稀释**令投资者权衡是否提前布局。

## 五、对A股/博主框架的意义

- **这份报告的核心 = 美股AI资本开支仍在扩张（利好上游硬件需求），但融资/稀释/回报可见度的担忧在累积**——这正是近期**美股megacap走弱、AI芯片见顶担忧**的根源，也是A股**光模块/CPO/AI硬件被带崩**的传导源头。
- 印证博主："**AI硬件（CPO/光模块）受美股牵制、只做美股企稳后的右侧**"——盯美股超大规模厂商Capex指引 + 融资动作，是判断A股AI硬件右侧的外部锚。
- **与第一份（A股一季报手册）配合看**：美股AI利润在硬件（capex扩张）↔ A股AI利润也转硬件（存储/材料）——全球AI利润池都在向上游硬件集中，但A股优势在**国产替代+反内卷涨价+出海**这些自主逻辑。
