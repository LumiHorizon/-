# 蒸馏｜2026-06-22 视频（非银金融暴涨：风格切换还是陷阱？）

## 一、核心结论

- **非银金融（券商）暴涨可能成新主线**：买券商≈买"科技影子基金"——科创板IPO**强制跟投**让券商吃科技股上市浮盈，叠加**长鑫即将上市**（若当日暴涨或上5万亿）、**国家队压制盘已基本卖光**、海外券商大涨映射、**PB百分位极低**。若今晚各大券商集体吹、叙事强化，**可能复制去年稳定币行情**（5月底涨到8–9月）。
- **但这是双刃剑/最大风险**：一旦非银叙事太强、资金从**光模块/液冷/PCB（海外AI硬件）切出**，这些方向会很危险，可能出现**"上证/创业板都好、但科技股被干得稀里哗啦"**的局面。
- **指数层面最担心**：上证逼近**4258前高**、今天放量**3.76万亿**——这个量过前高有**顶背离**风险；真放4万亿强行过去，则透支8–9月。

## 二、关键价位/信号

| 动作 | 触发条件 |
|------|---------|
| 指数能否过前高 | **上证4258**：看过关那天**量能**——健康放量过 vs 量过不去则"出大问题"；放4万亿一举过去难度大 |
| 非银叙事强化 | **今晚各大券商会不会集体出来吹**（一旦集体催→类比去年稳定币主线） |
| 科技危险信号 | 非银叙事确立 → **光/液冷/PCB 大资金出逃**（坚持光的大资金会嗅到味道出来） |
| 主线验证 | **主线不会套你超过3天**（套超3天=不是主线，参照算力租赁） |
| 强势科技上车点 | **回踩20日均线** |
| 明日盘面 | 券商强 + 收十字星/小阴线=好；**若继续猛搞→AI硬件会惨** |

## 三、行动清单

**优先做**
- **已有利润垫 + 已配一半防御（券商/保险）→ 不必再加仓，持有**（躺着收益不比科技弱）
- **全在科技/进攻盘的 → 看券商龙头**（可能走出趋势）；**保险不用追**（已高开）
- **锂电可配**：**恩捷股份（002812 深主板）隔膜高看一眼**（上周四跌得深）；宁德时代（300750 创业板）若起来行情不差
- **轻指数、重个股**：现在是"先有趋势、后有连板加速"，不用过去涨停启动的短线打法

**条件性做（若X→则Y）**
- 若非银叙事强化（券商集体吹）→ 可能成主线，**有利润垫可搞一把**，打仓位等（主线不套超3天）
- 若强势科技**回踩20日均线** → 可接
- 若资金切换、光出现调整 → 对持有的光/AI硬件**做T、留足安全垫**

**规避**
- **光模块/液冷/PCB（海外AI硬件）**：非银叙事确立则大资金出逃，**这个节骨眼要小心**，比上一轮更凶险
- **没利润垫/亏钱的别追高**——很多品种已涨十几到三四十个点，别两头挨打，找相对安全的争取下一轮
- **商业航天**：7月有催化（回收火箭大概率成），但**成只是第一步，警惕利好兑现变利空**

## 四、可复用框架

### 规律1：券商=科技的"影子基金"
- 信号：科创板IPO强制跟投（含未盈利企业），创业板四类必须跟投
- 含义：科技股上市浮盈进券商业绩，本轮AI牛市头部券商是境内外上市核心中介→买券商等同买科技
- 应对：科技牛市中后段，券商有补涨+业绩双逻辑（叠加长鑫上市、国家队卖压出清、PB极低）

### 规律2：风格切换的危险——非银强则AI硬件失血
- 信号：非银金融叙事被强化、资金大幅流入
- 含义：坚持光/AI硬件的大资金会嗅到味道出逃，造成"指数好但科技股崩"的撕裂
- 应对：非银一旦确立趋势，对光/液冷/PCB留安全垫、做T；存储因长鑫上市前相对安全

### 规律3：指数逼近前高看量能（顶背离风险）
- 信号：上证近4258前高、量能3.76万亿
- 含义：这个量过前高易顶背离；要么放4万亿一举过（难），要么砸有色/科技/权重压指数
- 应对：盯过关那天的量能质量，不健康放量过反而危险

### 规律4：主线不套超3天
- 信号：打进仓位后被套的时长
- 含义：真主线行情不会套你超过3天；套超3天大概率不是主线
- 应对：有利润垫可博，用"3天"做主线证伪的纪律线

### 规律5：先趋势、后连板（轻指数重个股）
- 信号：现在是趋势走出辨识度→再涨停加速（去年是涨停启动→后走趋势）
- 含义：连板是趋势加速的确认，不是启动点
- 应对：按当前风格选有趋势的个股，别用旧的涨停打板思路

### 规律6：强势股的上车点=20日均线
- 信号：均线已梳理完、动不动干到天上的强势科技股
- 含义：这类不好上车，回踩20日均线是相对好的位置
- 应对：等回踩20日均线接；中报方向均线未梳理完，7月中旬前上车机会更多

- **躺平就按每周一视频配置**（含时间窗口）；想精进看技术教学（股市33节课）。
- **不能"既要躺又要每天赚钱"**：涨10%+后回调正常，别上周四闹、这周一又不说话；交易不可能天天赚。
- **时间节点**：现在到**7月底是很好的做多窗口**（要说主升就这段）；**国庆后到11月较危险**；7月底前观点不变。
