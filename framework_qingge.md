## 零、认知层级（最底层）

> 博主输出优先级：**能不能做 > 怎么做 > 做什么**

### 市场本质：反身性，非均衡
- 参与者认知反映现实，同时通过行动改变现实——两者互为因果
- 价格不是钟摆式回归均衡，而是**路径依赖的正反馈过程**
- A 股定位：**弱势有效市场**（技术/基本面分析仍有超额收益空间）
  - 对政策利好**反应过度**（沾边就炒 → A杀 → 高位套牢）
  - 对财报业绩**反应不足**（超预期当日小涨 → 后续 1-3 个月才兑现）
- AI 本身没有泡沫，泡沫在于人类对 AI 的**集体想象**

### 时代底层逻辑变迁
| 旧秩序（2025前） | 新秩序（安全优先） |
|---|---|
| 效率优先，零库存快周转 | 不是「需要多少」而是「你能抢到多少」 |
| 算法/数字资产享溢价 | 矿山/电网/算力不可一夜建成 → 实物稀缺 |
| 全球化分工 | 供应链安全>效率，囤货=企业生存命脉 |

**当前周期定位**：康波第六波（AI）基建投入期 = 铁路时代的「铁路+电网」= AIDC+配套电力

## 一、情绪周期（核心能力）

> 博主最核心输出。情绪判断永远是第一优先级，早于推荐股票。

### 阶段划分与操作映射

| 情绪阶段 | 特征信号 | 操作 |
|---|---|---|
| **拐点酝酿** | 最高板被压缩到4板，连板晋级率极低 | 观望，不打飞刀 |
| **拐点确认** | 情绪由跌转涨的第一天 | 轻仓介入，等方向 |
| **上升第1-2日** | 首次分歧（高开回落）是洗盘 | 低吸主线核心，持股为主 |
| **上升第3日+** | 主线核心继续强势，资金大量流入 | 积极持仓，不追高 |
| **亢奋期尾部** | 指数创历史记录走势（如10连阳） | 只做最强前排，退潮不会A杀会横盘 |
| **退潮确认** | 跌停家数>20家/连板高度压缩至4板以内 | 降仓位聚焦中军龙头 |

### 关键信号清单
- **人民币背离信号**：人民币贬值 + A股继续涨 = 减仓预警，传导有1-2周时间差
- **90分钟顶部钝化 + 高9结构**：明确减仓信号，调整周期约8个交易日
- **高位放量阴线**：无条件先走
- **VIX恐慌飙升**：孕育年度级别大主线的买入窗口

### 周内节奏规律（连续5周验证）
- 周一情绪好赚钱效应强 → 正常操作
- **周三降仓位**，周五尾盘加回
- 周四周五情绪差亏钱效应大 → 规避重仓

## 二、宏观分析框架

### 全球宏观传导链
```
油价上涨
  → 推高通胀预期
    → 推高美债收益率
      → 压制成长股估值 → AI/科技股承压
      → 打击能源进口国（日本等）
        → 经常账户恶化 → 日元/亚洲货币贬值
          → 美元走强（灾难货币）
            → 日元若失守 → 套利交易逆转 → 全球风险资产同步抛售
```

### 关键监控指标

| 指标 | 当前关注 | 含义 |
|---|---|---|
| 十年期美债 | 4.4% 是全球资产定价锚 | 突破则股/金/汇全面重估 |
| 油价 | 120美元生死线（持续2月以上才改变框架） | 80→108→120三个情景 |
| 人民币中间价 | 每日9:30发布，与A股背离 = 减仓信号 | 领先指标，1-2周时间差 |
| 信用利差 | 持续扩张 = 风险情绪恶化 | 每日监控 |
| 比特币 | 风险偏好代理指标 | 参考用 |

### A股结构性支撑（宏观悲观时不轻言看空）
- 国家队从「托底」转向「管节奏」：制造慢牛，防2015悲剧重演
- 险资/ETF被动接盘提供自主承接力
- 3000点是左尾底部，慢牛里最大风险是被短期波动吓出场

### 跷跷板结构
外部（出海/外需）与内部（内需政策）交替进退，不同向。需分仓配置，动态再平衡。

## 三、选股框架

> 底层路径：**宏观判断 → 主线识别 → 产业链 → 具体标的**。图形指标只是确认工具。

### 选股三不做
1. 不根据图形涨幅榜从几千只中筛选
2. **不提前埋伏主线**（埋伏的对象是产业逻辑，不是主线）
3. 不看游资小作文做中长线（题材梳理是信息，不是买入信号）

### 主线识别三步
1. **大盘修复**：普跌转普涨，探底回升
2. **Price In消化**：坏消息不再有杀伤力
3. **连板梯队重建**：高标打出高度（先锋高标+容量中军+后排补涨）

**区分一波流 vs 大主线的关键点：分歧之后**
- 一波流：分歧后A杀
- 大主线：分歧后N波上行，再度加强（持续2-3月，核心涨幅通常翻倍）

### AI四象限（当前持仓框架参考）
| 象限 | 代表行业 |
|---|---|
| **受益区**（核心) | 铜铝/半导体/PCB/光模块/电力设备/电池/化工 |
| 堡垒区 | 高端白酒/国防/电网/港口（AI难替代） |
| 重塑区 | 法律/资管/高端教育（AI驱动但难完全替代） |
| 受损区 | 软件外包/广告/客服/翻译（AI直接替代） |

### 涨价逻辑（永恒可炒）
- 卖价涨10% → 成本不变 → 利润翻倍（+100%）
- 适用场景：存储芯片、化工、资源品
- 操作：加入硬件渠道群获取第一手涨价信息

### 估值工具优先级
**PE > PEG > PS > 市概率**（可靠性递减）
- PE失效（亏损/密集研发）时用PS，但必须配合毛利率+增速
- PS也失真时用「成为龙头的概率」（如寒武纪=英伟达市值×概率）
- PB适用五大行业（银行/保险/券商/地产/周期），看历史百分位而非绝对值

## 四、仓位管理

### 核心框架：80% 贝塔 + 20% 阿尔法
- **80%**：机构抱团票（如中际旭创），吃市场贝塔，大盘系统性风险时才动
- **20%**：主线题材和短线，拉高收益波动率
- 平时情绪波动只动20%；地缘冲突/宏观恶化才动80%

### 建仓节奏
- **大仓位建仓点**：趋势确立后满仓介入（不是底部埋伏）
- 先小仓位介入 → 小跌小买大跌大买 → 熟悉后才能做T
- 新手建议3-4个票，不要超10个

### 加减仓触发条件

| 触发信号 | 操作 |
|---|---|
| 情绪拐点确认 | 加仓（从持币转为持股为主） |
| 上升趋势确立 | 满仓 |
| 90分钟顶部钝化+高9 | 明确减仓 |
| 高位放量阴线 | 先走 |
| 大盘系统性风险（地缘冲突等） | 动80%大仓位 |
| 连续亏损 | 强制休息，等拐点再出手 |

### 牛市操作纪律
**不要卖掉自己没涨的去追已涨起来的**
主线内部轮动快，没涨≠逻辑不好，只是还没轮到。在熟悉的池子里选2-3个细分支，分歧时观察承接，高潮时不盲目追涨。

## 五、风控逻辑

### 第一性原理：动态风控，非固定止损线

**产业逻辑标的止损**：逻辑破坏才走（业绩不及预期），价格波动不构成止损理由

**短线标的退出信号**：
- 高位放量阴线 → 无条件先走
- 该强不强（平开/低开） = 已经弱了 = 危险信号，及时应对
- 连板标的当天没涨停必须走

### 黑天鹅应对
- 美债利率突破关键位 → 降仓等确认
- 地缘冲突突发 → 先降后观察（三天内通常有下跌+反弹机会）
- 外盘流动性挤压导致A股回撤 → 视为**买入机会**，保留抄底能力

### 心态风控
- 贪婪/恐惧的根源：**仓位超出承受范围**。降仓位，心态自然平复
- 操作成瘾自测：行情好能赚，行情差就大亏 → 降低决策频率
- 降低决策频率的方法：找一周中盈亏比最高的1-2个机会，其余时间观望

## 六、技术分析体系（辅助确认用）

### 七轨布林线（博主自研）
```
MID = MA(C, 20)；DEV = MA(STD(C,20), 5)
顶轨 = MID + 3×DEV   ← 极端超买，无条件减仓
一轨 = MID + 2×DEV   ← 中度超买，考虑减仓
二轨 = MID + 1×DEV   ← N字回踩买点（强势股买入区）
中轨 = MID            ← 强势股沿此持有
四轨 = MID - 1×DEV   ← 观望
五轨 = MID - 2×DEV   ← 分批建仓
底轨 = MID - 3×DEV   ← 极端超卖，大概率反弹
```

**三种经典形态**：
- 强势股：底轨→顶轨→回调到二轨再攻
- 收敛股：横盘带宽收窄→任意涨几天破顶轨
- N字启动：首板→回踩二轨→第二波

### 量价关系（优先级高于布林线）
| 信号 | 操作 |
|---|---|
| 高位放量阴线 | 先走，不犹豫 |
| 不放量A杀 | 板块有承接，可等反弹 |
| 放量A杀 | 板块弱，资金出逃，回避 |
| 偏离5日均线过远 | 警惕均值回归 |

### 顶部形态识别
- **90分钟顶部钝化**：指数连续创新高后出现，调整周期约8个交易日
- **高位放量长阴线**：无条件清仓信号（博通财报利空案例）
- **上吊线/流星线**：高位见顶预警，需结合前方走势确认

## 八、博主画像（理解输出的前提）

| 维度 | 特征 |
|---|---|
| 身份 | 产业分析+情绪周期博主，非选股老师 |
| 核心价值 | 每日情绪判断和盘面解读（"情绪时钟"） |
| 实战风格 | 给出情景应对矩阵（如果A则B），可执行 |
| 信息处理 | 每天约100场电话会议+上百份研报，筛选后录视频 |
| 对散户建议 | 先花6个月感受情绪再学方法；先就业再炒股 |
| 角色定位 | 研究员+风控；用户是基金经理，自己对账户负责 |
